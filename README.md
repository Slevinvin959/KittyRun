# KittyRun 🐱

2D-платформер про кота, который ищет миску с молоком.

## Управление
- ⬅️ ➡️ — движение
- ␣ — прыжок
- A — атака

## Установка и запуск
1. Установи библиотеку arcade:
pip install arcade

text
2. Запусти игру:
python kitty_run.py

text

## Технологии
- Python
- Arcade
- Программная отрисовка графики примитивами

## Уровни
- **1 уровень** — обучение, простые платформы
- **2 уровень** — сложнее, больше врагов
- **3 уровень** — вода вместо пола, нужно прыгать по платформам
